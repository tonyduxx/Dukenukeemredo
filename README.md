# Hydra Heros

A modern reimagining of the classic Duke Nukem 3D game, built as a web-based game with character selection.

## Features

### Character Selection
Choose from three unique heroes, each with their own characteristics:

- **DUXX** - The Tech Specialist
  - High accuracy, moderate damage
  - Perfect for precision gameplay

- **VAN** - The Assault Expert  
  - Balanced stats, versatile fighter
  - Great all-around character

- **GRIZZ** - The Heavy Gunner
  - High damage, slower movement
  - Best for aggressive playstyles

### Game Screens

1. **Title Screen** - Main menu with classic game options
2. **Character Selection** - Choose your hero before starting
3. **Game Screen** - The main gameplay area (framework ready for expansion)
4. **Options Screen** - Game settings and preferences
5. **Credits Screen** - Attribution to original Duke Nukem 3D

## How to Play

1. Open `index.html` in your web browser
2. Click "NEW GAME" from the main menu
3. Select one of the three characters (Duxx, Van, or Grizz)
4. Click "START GAME" to begin playing

## Controls

- **Mouse**: Navigate menus and select options
- **Escape Key**: Return to previous screen or main menu
- **Keyboard Navigation**: Use arrow keys in menus

## Technical Details

- Built with modern HTML5, CSS3, and JavaScript
- Responsive design that works on desktop and mobile
- Uses Google Fonts (Orbitron) for a sci-fi aesthetic
- CSS animations and transitions for smooth gameplay
- Modular screen system for easy expansion

## Inspired By

This game is inspired by the Duke Nukem 3D source code released by 3D Realms. The original game structure and menu system influenced the design, but this is a completely new implementation built for modern web browsers.
